📁 项目说明
-----------
本系统包含三个主要页面：
- P0 宏观溯源：迁徙地图 + 15分钟回放
- P1 全局态势：5G-A资源网格 + Co-Pilot
- P2 场内微观：✅ 当前页面（已完成）

🚀 快速启动
-----------
1. 确保已安装 Node.js (v18+)
   下载地址: https://nodejs.org/

2. 双击运行 "start.bat"

3. 首次运行会自动安装依赖（约需2-3分钟）

4. 浏览器自动打开 http://localhost:3002

📋 系统要求
-----------
- Windows 10/11 或 macOS/Linux
- Node.js v18.0 或更高版本
- Chrome/Edge/Firefox 现代浏览器
